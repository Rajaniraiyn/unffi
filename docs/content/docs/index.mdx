---
title: UnFFI
description: Universal FFI for Bun, Deno, and Node.js.
---

UnFFI lets JavaScript call native C ABI libraries with one TypeScript schema. Use it for your own shared libraries, generated bindings, or shipped OS modules such as CoreFoundation, libc, and kernel32.
